[Documentation](https://accesspressthemes.com/documentation/ap-custom-testimonial/) | [Support](https://accesspressthemes.com/support) | [Demo](http://demo.accesspressthemes.com/wordpress-plugins/ap-custom-testimonial/) | [Premium Version](https://accesspressthemes.com/wordpress-plugins/ap-custom-testimonial-pro/)

Create simple, responsive, dynamic testimonials for your site. Very easy to use via shortcode anywhere in the site.

= Free Features: =

* <strong>Five pre available template designs</strong> - You can choose among two different layouts along with five nice template designs.
* <strong>Simple settings</strong> - Simple settings with multiple usability.
* <strong>Static Shortcode</strong> - You can display the testimonial easily anywhere by using shortcode [ap_custom_testimonial].
* <strong>Dynamic Shortcode Generator</strong> - You can generate dynamic shortcode with additional options as you need and as many as you need.
* <strong>Responsive</strong> - Easy compatible with smaller devices.
* <strong>Custom Design</strong> - Additional settings to design your own custom testimonial layout by enabling it while generating shortcode implementing your own typography,background-color,image and other settings.
* <strong>Dynamic slider control option</strong> - While generating shortcode for slider layout, you can choose custom slider controls for each distinctive shortcode.
* <strong>Dedicated Support</strong>
        -Email and forum support.
* <strong>Free updates</strong>
        - Free updates for plugin.

= Usage =

* Static Shortcode:
You can use default shortcode [ap_custom_testimonial] that display the default list testimonial layout with template 1 design.

* Dynamic Shortcode:
You can also use Dynamic shortcode Generated by Dynamic shortcode generator page in plugin.

== Installation ==
1. Unzip ap-custom-testimonial.zip
2. Upload all the files to the /wp-content/plugins/ap-custom-testimonial
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on AP Custom Testimonial menu in Wordpress left admin menu.
